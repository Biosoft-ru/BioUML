FTP upload

Here you may upload the file from an FTP server directly to the system.

Parameters
  • FTP file URL – FTP file address in correct format like ftp://user:password@ftp.example.com/path/file.txt
  • Target – target folder in the tree to put the uploaded file to.
  • Importer for uploaded file – file format. Please select it carefully.
  • Importer properties – additional format-related options if applicable.

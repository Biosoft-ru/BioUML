Building Flux Balance DataTable

This analysis constructs FBC data table for Flux Balance Constraint analysis. This table contains information about fluxes (reactions): lower and upper bound, objective function coefficients. If input diagram is sbml-fbc, all data about fluxes is already stored in it and analysis just convert this information into the result table. If information for some fluxes is missed or diagram is not sbml-fbc default values specified in parameters will be added to the result table.

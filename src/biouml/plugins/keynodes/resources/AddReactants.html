Results of effector/regulator search, longest chains finder and results of other similar analyses contains only essential elements, so visualizations of such results, which are usually used as scaffold of the signalling paths, contains only necessary elements. But if one would like to model all the dynamics or all the fluxes, diagram which contains all reactants/products fro each reaction is needed.
This analysis adds missing reactants/products to reactions of existing diagram.

Visualize results of drug target search analysis

Goal:

Display signalling networks as diagram.

Parameters:

  • Analysis result — a result of %analysis:Regulator search% or %analysis:Effector search% analyses
  • Rank column — a name of a column in analysis result table, that will be used as rank, i.e. table will be sorted in descending order by this column
  • Number of top ranking molecules — signalling networks corresponding to selected top lines will be displayed as diagram
  • Output name — result diagram path

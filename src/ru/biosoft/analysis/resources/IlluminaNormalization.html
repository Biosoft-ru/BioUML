Illumina Normalization

Normalization of Illumina files based on the functions of the Bioconductor LUMI package.

Normalization will be done using the lumiExpresso function which starts from raw Illumina probe intensities and returns expression values.

For the detailed description of the LUMI functions see the LUMI reference manual.

Note: Before the Illumina Normalization is started you need to install R and Rserve on your computer. For more information on installing, please visit R-project website.

Note: Duplicated identifiers will be modified.

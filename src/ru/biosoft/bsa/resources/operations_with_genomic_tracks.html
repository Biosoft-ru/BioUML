What is genomic track?
A genomic track consists of a set of genomic sites. Each genomic site has a genomic position, such as chr1:100-200, an optional strand (+ or -) and a set of optional properties, such as (score, name).
View tracks
In BioUML genomic tracks can be viewed eihter as a table or in genome browser. Tracks can be associated with some genome build (sequences collection) in which case you can double click on the track to open it in the genome browser. Alternatively you can first open chromosome in the genome browser, and then drag-and-drop the track on it. Tracks can be opened as a table document using the 'Open as table' context action. In this case a list of all sites in the track will be displayed in tabular form with the ability to sort, filter and so on.
Obtain tracks
Genomic tracks can be imported into BioUML from many different file formats:
  • BED format (*.bed)
  • Gene Transfer Format (*.gtf)
  • General Feature Format (*.gff)
  • Interval format (*.interval)
  • SAM or BAM alignment file (*.sam, *.bam)
  • VCF format (*.vcf)
Alternatively, track can be created from existing table with Convert table to track analysis.

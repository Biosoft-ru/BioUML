Jupyter ноутбуки

Jupyter notebook/hub - широко используемый инструмент для интерактивного воспроизводимого анализа и визуализации данных. Он поддерживает использование множества языков программирования. Для этого используется понятие kernel - для каждого ноутбука на сервере запускается указанный kernel, который собственно и выполняет анализ и визуализацию данных, взаимодействуя с веб-интерфейсом (ноутбуком) через заданный API и передачу данных в JSON формате.
Для этого были разработаны библиотеки на языках программирования R и Python 3:
  • rbiouml (https://cran.r-project.org/web/packages/rbiouml/index.html) - библиотека для работы с BioUML с помощью написания кода на языке программирования R.
  • pybiouml (https://pypi.org/project/pybiouml/) - библиотека для работы с BioUML с помощью написания кода на языке программирования Python 3.

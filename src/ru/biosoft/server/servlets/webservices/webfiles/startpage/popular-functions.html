Popular functions

  • Operations with tables
    • Annotate table [?]
      Annotate table with additional fields from database
      (gene name, description and so on)
    • Convert table [?]
      Convert identifiers of a table
    • Join tables [?]
      Intersect or join two or more tables
    • Venn diagram [?]
      Plot venn diagram for given tables
  • Operations with tracks
    • Annotate track with genes [?]
      Add properties to each track site describing genes where this site falls
    • Gene set to track [?]
      Convert genes to the track of promoters
    • Track to gene set [?]
      Get all the genes that are near the track sites
    • Process track with sites [?]
      Enlarge/shrink sites on the track, merge overlapping sites or remove too short sites
    • Track statistics [?]
      Gather various statistics about track or FASTQ file
    • Mutation effect [?]
      Determine effect of mutation on protein
    • Remove overlapping sites [?]
      Construct subset of input track so that no sites overlap in the output track
  • Statistical methods
    • CRC clustering [?]
      Chinese restaurant clustering
    • K-means clustering [?]
      Genes are grouped into clusters so that those in one cluster exhibit maximal similarity
